git 명령어 실습 정리

1. git 푸시 방법

  1. git add .
  2. git commit -m "commit message"
  3. git push

2. git 브랜치 생성 커밋, 마스터 머지

  1. git branch
  2. git branch develop
  3. git checkout develop
  4. 파일 수정 추가 후 1번 한 사이클 돌기
  5. git checkout master
  6. git merge develop
  7. git push

3. 커밋 내용 수정하기

  • git commit --amend

4. 브랜치 삭제하기

  • git branch -d develop

5. git log확인

  • git log

6. git 상태 확인

  • git status
