git 명령어 실습 정리
	

1. git 푸시 방법

git add .
git commit -m "commit message"
git push


2. git 브랜치 생성 커밋, 마스터 머지

git branch
git branch develop
git checkout develop
파일 수정 추가 후 1번 한 사이클 돌기
git checkout master
git merge develop
git push



3. 커밋 내용 수정하기

git commit --amend



4. 브랜치 삭제하기

git branch -d develop



5. git log확인

git log



6. git 상태 확인

git status
